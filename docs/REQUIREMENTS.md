# QuickBG - 在线图片背景移除工具 需求文档

> 版本：v1.0.0 | 更新日期：2026-04-03

## 1. 项目概述

### 1.1 产品定位

QuickBG 是一款 **AI 驱动的在线图片背景移除工具**（Image Background Remover），面向电商卖家、内容创作者、设计师和普通用户，提供一键式智能抠图服务。用户无需安装任何软件，打开浏览器即可使用。

### 1.2 核心价值

- **零门槛**：无需注册即可体验，打开网页直接使用
- **快速度**：AI 自动处理，平均 5 秒内完成背景移除
- **高质量**：边缘精细处理，支持最高 4K 分辨率输出
- **多格式**：支持 PNG 透明背景、JPG 自定义背景色下载

### 1.3 目标用户

| 用户类型 | 使用场景 | 核心诉求 |
|---------|---------|---------|
| 电商卖家 | 商品白底图、主图制作 | 批量处理、高效出图 |
| 内容创作者 | 社交媒体图片、封面制作 | 快速处理、效果自然 |
| 设计师 | 设计素材预处理 | 高质量输出、灵活格式 |
| 普通用户 | 证件照换底、头像制作 | 简单易用、免费体验 |

## 2. 功能需求

### 2.1 核心功能（P0 - MVP 必须实现）

#### 2.1.1 图片上传
- 支持拖拽上传和点击选择文件
- 支持格式：JPG、PNG、WebP
- 文件大小限制：10MB（免费版）/ 25MB（专业版）/ 50MB（企业版）
- 上传前实时预览，显示文件名和大小
- 输入校验：格式不合法或超限时给出明确提示

#### 2.1.2 AI 背景移除
- 调用 Remove.bg API 进行智能抠图
- 处理状态展示：上传中 → 处理中 → 完成/失败
- API 不可用时优雅降级（提示用户稍后重试）
- 错误处理：API 调用失败时展示友好错误信息

#### 2.1.3 结果预览与对比
- 棋盘格背景展示抠图结果（直观展示透明区域）
- 原图/结果左右滑动对比（滑块式对比组件）
- 支持缩放查看细节

#### 2.1.4 图片下载
- **PNG 下载**：透明背景，保留原始分辨率
- **JPG 下载**：支持自定义背景色，内置常用色板（白、黑、灰、蓝等）
- 下载文件名自动带 `_no-bg` 后缀

### 2.2 用户系统（P0）

#### 2.2.1 注册与登录
- 邮箱 + 密码注册/登录
- 表单校验：邮箱格式、密码长度（≥6位）、密码确认一致性
- 登录状态通过 localStorage 持久化

#### 2.2.2 用户仪表盘
- 展示账户信息：邮箱、当前套餐、剩余额度
- 处理历史记录列表：文件名、大小、处理时间、状态
- 快捷入口：一键跳转到抠图工具页

### 2.3 定价体系（P0 - 前端展示）

| 功能 | 免费版 | 专业版 ¥29/月 | 企业版 ¥99/月 |
|-----|-------|-------------|-------------|
| 每月处理次数 | 5 次 | 100 次 | 无限 |
| 输出质量 | 标准 | 高清 | 4K 超高清 |
| 下载格式 | JPG/PNG | 全格式 | 全格式 |
| 批量处理 | ❌ | 最多 5 张 | 无限 |
| API 访问 | ❌ | ❌ | ✅ + 优先支持 |

### 2.4 增强功能（P1 - 后续迭代）

- [ ] 批量上传与处理（拖入多张图片）
- [ ] 批量下载（ZIP 打包）

## 3. 非功能需求

### 3.1 技术栈

| 层级 | 技术选型 | 说明 |
|-----|---------|------|
| 前端框架 | Next.js 16 (App Router) | React 19 + TypeScript |
| 样式方案 | Tailwind CSS 4 | 原子化 CSS，高效开发 |
| 图标库 | Lucide React | 轻量级 SVG 图标 |
| 抠图 API | Remove.bg API | 第三方 AI 抠图服务 |
| 部署方案 | Vercel / 自托管 | 环境变量管理 API Key |

### 3.2 性能要求

- 首屏加载时间 < 2s（LCP）
- 图片上传响应 < 1s（本地处理）
- API 抠图处理 < 10s（依赖 Remove.bg 服务）
- 页面交互无感知延迟（< 100ms）

### 3.3 安全要求

- API Key 通过服务端环境变量存储，不暴露给客户端
- 用户上传图片不持久化存储，处理完成后立即释放
- 全站 HTTPS 加密传输
- 防止 XSS 攻击（输入校验 + 输出转义）

### 3.4 兼容性

- 浏览器：Chrome 90+、Firefox 90+、Safari 15+、Edge 90+
- 设备：桌面端优先，平板端兼容，移动端基本可用
- 分辨率：1366×768 ~ 4K

## 4. 页面结构

```
/                    → 首页（Hero + 功能展示 + 定价预览 + CTA）
/remove              → 在线抠图工具（上传 → 处理 → 对比 → 下载）
/pricing             → 定价详情（方案对比 + FAQ）
/login               → 登录页
/register            → 注册页
/dashboard           → 用户仪表盘（需登录）
/api/remove-bg       → 抠图 API（POST，接收 FormData）
/api/auth/login      → 登录 API（POST）
/api/auth/register   → 注册 API（POST）
```

## 5. 设计规范

### 5.1 视觉风格
- 主色调：Indigo (#4F46E5) → Purple 渐变
- 辅助色：Gray 系列中性色
- 圆角：组件统一使用 `rounded-xl` / `rounded-2xl`
- 阴影：`shadow-md` / `shadow-lg`，hover 时增强
- 字体：系统默认字体栈（`antialiased`）

### 5.2 交互规范
- 按钮点击：`active:scale-[0.98]` 微缩放反馈
- 页面切换：平滑过渡，无闪烁
- 拖拽上传：拖入时边框高亮 + 背景色变化 + 图标切换
- 图片对比：滑块式拖拽，指针捕获（Pointer Events）
